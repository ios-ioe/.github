This is the **community health files repository** for [Ignite Open Source (IOS)](https://github.com/ios-ioe).

## What is this repository?

This special `.github` repository contains default community health files that apply to all repositories in the IOS organization that don't have their own.

## Contents

| File | Description |
|------|-------------|
| [CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md) | Community standards and expected behavior |
| [CONTRIBUTING.md](CONTRIBUTING.md) | Guidelines for contributing to IOS projects |
| [profile/README.md](profile/README.md) | Organization profile displayed on our GitHub page |

## How it works

- Files in this repository serve as **defaults** for all IOS repositories
- Individual repositories can override these by adding their own versions
- The `profile/README.md` is displayed on our [organization's GitHub page](https://github.com/ios-ioe)
